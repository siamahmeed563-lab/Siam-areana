# ⚽ SIAM ARENA — Live TV & World Cup 2026

**Siam Arena** is a live TV streaming website for Bangladeshi channels and World Cup 2026 matches.

🌐 **Website:** [siamahmeed563-lab.github.io/Siam-areana](https://siamahmeed563-lab.github.io/Siam-areana)

---

## 📺 Features

- 🔴 Bangladeshi channels live streaming
- ⚽ World Cup 2026 all matches live
- 📅 Match schedule & fixtures
- 📊 Group standings
- 📡 40+ sports channels
- 📱 Mobile & TV friendly

---

## 📡 IPTV Playlist

Add this URL in Televizo or any IPTV player:

https://siamahmeed563-lab.github.io/Siam-areana/playlist.m3u

playlist.m3u updates automatically whenever channels.json is updated.

---

## 📂 Files

| File | Purpose |
|------|---------|
| index.html | Main website |
| channels.json | Channel list & stream URLs |
| playlist.m3u | IPTV Playlist (auto-generated) |
| generate_playlist.py | Generates playlist from channels.json |
| .github/workflows/ | Auto-update system |

---

## How to update channels?

Just edit channels.json — everything else updates automatically.
