# ⚽ Siam Arena – World Cup 2026

World Cup 2026 Live Coverage App

## Features
- 📺 Live TV (BTV, T Sports, Somoy TV + more)
- 📅 Match Schedule
- 📊 Group Standings
- 🔗 External streaming sites
- 📱 Installable PWA

## Deploy on GitHub Pages
1. Upload all files to a GitHub repository
2. Go to Settings → Pages → Branch: main → Save
3. Visit: `https://yourusername.github.io/siam-arena`

## API Key
Get a free API key from [football-data.org](https://www.football-data.org/client/register)
Then replace `YOUR_FOOTBALL_DATA_API_KEY` in `index.html`
